# 算法考点总结

> 每个算法擅长解决的问题

---

## 基础技巧

| 算法 | 擅长解决的问题 |
|------|--------------|
| **二分查找** | 有序数组中查找元素、寻找边界、最大化最小值/最小化最大值问题 |
| **双指针** | 有序数组两数之和、移除元素、链表找环、滑动窗口 |
| **滑动窗口** | 子串/子数组问题（最大和、最小覆盖、无重复字符） |

---

## 搜索与遍历

| 算法 | 擅长解决的问题 |
|------|--------------|
| **BFS** | 最短路径（无权图）、层序遍历、迷宫问题、单词接龙 |
| **DFS** | 路径问题、岛屿数量、树形遍历、排列组合、N皇后 |
| **回溯** | 全排列、组合总和、子集、棋盘问题、组合优化 |

---

## 基础数据结构

| 算法 | 擅长解决的问题 |
|------|--------------|
| **栈和队列** | 括号匹配、单调栈（下一个更大/小元素）、表达式求值 |
| **哈希表** | 快速查找、去重、两数之和、LRU缓存、字符计数 |
| **链表** | 反转、合并、环检测、删除倒数第N个节点 |

---

## 树形结构

| 算法 | 擅长解决的问题 |
|------|--------------|
| **二叉树** | 遍历、构建（前后+中序）、路径总和、验证BST |
| **线段树** | 区间求和/最大值、区间更新、动态 RMQ |
| **树状数组** | 单点更新+区间求和、逆序对、动态前缀和 |
| **Trie字典树** | 字符串前缀匹配、自动补全、敏感词过滤、IP路由 |

---

## 图论

| 算法 | 擅长解决的问题 |
|------|--------------|
| **并查集** | 连通分量、朋友圈、省份数量、冗余连接、MST |
| **最短路径** | Dijkstra: 非负权单源最短路径<br>Bellman-Ford/SPFA: 含负权边<br>Floyd: 多源最短路径 |
| **最小生成树** | Kruskal: 稀疏图、边排序<br>Prim: 稠密图、点扩展 |
| **拓扑排序** | 依赖排序、课程先修、任务调度、编译顺序 |

---

## 动态规划

| 类型 | 擅长解决的问题 |
|------|--------------|
| **线性DP** | 爬楼梯、打家劫舍、股票买卖、斐波那契 |
| **背包DP** | 0/1背包、完全背包、多重背包、组合优化 |
| **区间DP** | 矩阵链乘、戳气球、博弈论 |
| **树形DP** | 二叉树直径、树的独立集 |
| **状态压缩DP** | 旅行商、蒙德里安铺砖 |

---

## 字符串

| 算法 | 擅长解决的问题 |
|------|--------------|
| **KMP** | 字符串匹配、模式查找、重复子串 |
| **Sunday** | 字符串匹配（比KMP更快） |
| **Rabin-Karp** | 字符串哈希匹配、多模式匹配 |

---

## 贪心

| 问题类型 | 擅长解决的问题 |
|----------|--------------|
| **区间调度** | 最多不重叠区间、会议室安排、活动选择 |
| ** Huffman** | 最小成本合并、哈夫曼编码 |
| **跳跃游戏** | 能否到达终点、最小跳跃次数 |
| **加油站** | 环形路径可行性 |

---

## 排序

| 算法 | 擅长解决的问题 |
|------|--------------|
| **快速排序** | 原地排序、平均O(nlogn)、找第K大/小 |
| **归并排序** | 外部排序、稳定性要求、逆序对 |
| **堆排序** |  Top-K问题、优先队列、几乎有序数组 |
| **计数排序** | 整数范围较小、桶排序基础 |
| **基数排序** | 多位数整数排序、字符串排序 |

---

## 数学

| 技巧 | 擅长解决的问题 |
|------|--------------|
| **快速幂** | 大数幂模、矩阵幂、组合数计算 |
| **GCD/LCM** | 辗转相除、分数简化、最小公倍数 |
| **质数筛法** | 质数判定、质数表、唯一质因数分解 |
| **位运算** | 奇偶判断、交换变量、消除最低位1、找单独数 |

---

## 其他技巧

| 技巧 | 擅长解决的问题 |
|------|--------------|
| **前缀和** | 静态区间求和、子数组和为K、连续子数组最大和 |
| **差分数组** | 区间批量增减、航班预订、最终状态查询 |
| **分治** | 归并排序、快速排序、最大子数组、幂运算 |

---

## 选择指南

```
问题特征 → 推荐算法

有有序数组 → 二分查找
子串/子数组 → 滑动窗口
最短路径（无权）→ BFS
最短路径（非负权）→ Dijkstra  
存在负权 → Bellman-Ford
连通性检测 → 并查集
区间求和/更新 → 线段树/树状数组
前缀匹配 → Trie
依赖排序 → 拓扑排序
最优子结构 → DP
局部最优 → 贪心
遍历搜索 → DFS/回溯
```
