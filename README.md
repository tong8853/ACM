## 🚀 主要内容

### ✨ 核心算法

| 分类 | 包含内容 |
|------|----------|
| 🔍 **搜索** | DFS, BFS, 回溯 |
| 🌲 **数据结构** | 线段树、树状数组、并查集、堆 |
| 📊 **图论** | 最短路、最小生成树、连通分量 |
| 🔤 **字符串** | KMP、Trie、AC自动机 |
| 🔢 **数论** | 模运算、GCD、质数筛法 |
| 💡 **贪心** | 区间调度、哈夫曼编码 |

### 📝 模板文件

- `模板.cpp` - C++ 竞赛常用模板
- `模板.py` - Python 竞赛常用模板
- `模运算模板.cpp` - 模运算专用模板

## 📈 学习路线

```
入门 → 基础算法 → 数据结构 → 图论 → 高级专题
```

1. **入门阶段**: 洛谷基础题、简单模拟
2. **基础阶段**: 二分、贪心、排序、DFS/BFS
3. **进阶阶段**: 线段树、并查集、最短路
4. **高级阶段**: 网络流、字符串算法、数论
